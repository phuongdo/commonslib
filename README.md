## Technology Stack

  * Java 1.8
  * nem.core

## How to build
Make sure to clone NEMModules fork of nem.core and build.

```bash
git clone https://github.com/NEMPH/nem.core.git
cd nem.core
mvn clean install
```

build the nem-apps-lib after.

```bash
git clone https://github.com/NEMPH/nem-apps-lib.git
cd nem-apps-lib
mvn clean install
```
